# React Example

Waitlist form component using React hooks.

## Usage

1. Replace `YOUR_WAITLIST_KEY` in `WaitlistForm.jsx`
2. Import and use in your app:

```jsx
import WaitlistForm from './WaitlistForm';

function App() {
  return <WaitlistForm />;
}
```

## Files

- `WaitlistForm.jsx` - Form component with useState
