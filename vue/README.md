# Vue Example

Waitlist form component using Vue 3 Composition API.

## Usage

1. Replace `YOUR_WAITLIST_KEY` in `WaitlistForm.vue`
2. Import and use in your app:

```vue
<script setup>
import WaitlistForm from './WaitlistForm.vue';
</script>

<template>
  <WaitlistForm />
</template>
```

## Files

- `WaitlistForm.vue` - Single-file component with `<script setup>`
